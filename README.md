# software-Canva
Ana, Davi

A EPTG Soluções Ágeis Ltda é uma startup de tecnologia da informação responsável pelo desenvolvimento do Smart Canvas, um software voltado para gestão estratégica ágil. Criado em agosto de 2024, o projeto vem apresentando crescimento constante e atualmente conta com aproximadamente 500 espaços de trabalho ativos, com uma média de 15 novos clientes por dia. A plataforma é multiplataforma e há planos de expansão para outros mercados ao longo de 2025.

O Smart Canvas foi desenvolvido utilizando a tecnologia Low Code da plataforma Bubble. No momento, um dos principais desafios da empresa é encontrar profissionais que já possuam conhecimento ou experiência nessa ferramenta. Por isso, a EPTG busca formar novos colaboradores capacitados para atuar no desenvolvimento e aprimoramento do sistema, especialmente na área de integrações.

O foco atual do projeto é estruturar e disponibilizar uma API organizada e bem documentada, permitindo que outras plataformas possam se conectar facilmente ao Smart Canvas. Entre as integrações previstas estão aplicações como Slack, Microsoft Teams, Google Agenda, Google Drive, Conta Azul, RD Station e GitHub.
